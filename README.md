1️⃣ What is the difference between var, let, and const?

var: This is Function-scoped। So if we declare in function by var, it can be accessed in everywhere of that function. But it is not Block-scoped.

let ও const: Both are Block-scoped। So, they are located in second bracket and do their function only in second bracket. We can't get them of those second brackets.

In javascript (ES6+), Spread Operator means three dot (...). The main function of spread operator (...) divided of Iterable in different parts.

3️⃣ What is the difference between map(), filter(), and forEach()?

map(): it is done defined function of every parts of array & It's return new array. No changing in main array.

filter(): It selects different parts of array based on condition and return new array. Which will fill up condition, will stay in new array.

forEach(): it runs function of every single parts of array but it does not return new array. It's result always undefined

4️⃣ What is an arrow function? 

Arrow Function in modern javascript (ES6) is a method that is easy and precised. Rather than traditional function declaration it simplifies the code and helps to read easily.

5️⃣ What are template literals?

Template Literals in javascript (ES6) is modern and strong method of string writing. Generally we write simple string with single quote (' ')or double quote (" ")but template literals is written with Backtick (`) sign. It helps the code to be dynamic and clear.
